Scripts in this directory works with data files in the format defined for The Cancer Genome Atlas.

tcga.pl converts output of the profiler into .mirna.quantification.txt and .isoform.quantification.txt files.
create_adf.pl creates a miRNA reference ADF file based on miRBase data.
expression_matrix_mimat.pl creates an expression matrix of miRNA mature strands by reading a Level 3 archive and a reference ADF
