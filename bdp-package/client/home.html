Profiling miRNA species from small RNA-seq sequence datasets


Introduction

This application summarizes miRNA expressions, identifies differentially expressed miRNA, and functional enrichment analyses for downstream target genes from small RNA sequence datasets (fastq.gz files). The analysis workflow follows the miRNA analysis pipeline of Genomic Data Commons (GDC). We also performed the multiQC to help users to understand the sequence quality. For miRNA expression profiles, we applied the BCGSC miRNA profiling pipeline generates miRNA expression profiles as documented in the GDC. For the additional post-analysis, expression profiles are generated by the DESeq2 to generate differentially expressed miRNA species. Then, target genes of these positively and negatively expressed miRNAs are extracted based on the pre-computed miRNA targets from TargetScan. The final results are both in the formats of PDF and Excel. We also made an integrated web user interface to display results interactively. Besides, this application is also used to demonstrate that BDP package developers can design graphical web interfaces for their workflows, so that users can have friendly interfaces to execute pipelines and visualize results interactively.
