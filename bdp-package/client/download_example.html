Introduction

The demo sequencing datasets to be downloaded is from a published study (Santangelo, L., Giurato, G., Cicchini, C., Montaldo, C., Mancone, C., Tarallo, R., Battistelli, C., Alonzi, T., Weisz, A., and Tripodi, M. (2016). The RNA-Binding Protein SYNCRIP Is a Component of the Hepatocyte Exosomal Machinery Controlling MicroRNA Sorting. Cell Rep 17, 799-808.). These small RNA-seq datasets were deposited on the ArrayExpress, and the accession is E-MTAB-4502. With these datasets, we can compare exosomal and cellular miRNA expressions and perform enrichment analysis for the differentially expressed miRNAs. For the demo purpose, only a part of the original datasets are downloaded via this web page.

Clicking the folowing button to create a folder record and the automatic process will download the above mentioned 18 files into this folder.

*Note that when the downloading process is failed for some reasons like no internet connections or the unavailable ftp.sra.ebi.ac.uk service, you may also manually download those files listed from here: https://www.ebi.ac.uk/arrayexpress/files/E-MTAB-4502/E-MTAB-4502.sdrf.txt.
